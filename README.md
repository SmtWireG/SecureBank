## üîê Features

- **Secure Login** with masked password input (using `getch()`)
- **Account Operations**:
  - Balance Inquiry  
  - Deposit / Withdraw  
  - IBAN-based Transfer (with dual-logging)
- **Transaction History Log**  
- **Object-Oriented C++ Architecture**
